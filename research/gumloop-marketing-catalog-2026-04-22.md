## Signal from the full catalog (vs. the 13-template preview)

**New themes the preview missed:**

1. **Content repurposing is a dominant workflow** — 7+ templates
   cover "turn X into Y" (blog→LinkedIn, YouTube→blog,
   article→viral posts, blog→shareable insights). This is a
   pivotal solo-founder skill.
2. **Competitive intel spans every function** — ads, SEO, content,
   product positioning. Each agent needs a competitor lens; HoM
   aggregates.
3. **Multi-platform social** (X, LinkedIn, Reddit, Instagram,
   TikTok) is heavily represented. Social agent must be
   multi-platform, not LinkedIn-only.
4. **Customer case study drafting** is its own artifact —
   distinct enough to warrant a dedicated skill.
5. **Market segmentation / buyer persona** is a named template
   (not just a generic "ICP" skill). The HoM `profile-icp` skill
   should produce a proper persona doc, not a one-liner.
6. **Sales call intelligence** for product marketing — mining
   calls for positioning signals. HoM skill.
7. **Blog post generation** is a template on its own — the skill
   needs to stand alone, not be hidden inside `plan-content-calendar`.
8. **Reddit reply / community engagement** — Social agent needs
   a `draft-community-reply` skill (Reddit first, generalizable).
9. **Video → text repurposing** (YouTube) — the repurpose skill
   should accept video sources, not just text.
10. **GEO audit** shows up twice (report + flow) — validation
    that AI-search visibility is a real emerging category.
